About Project:
--------------

This project is build using following:
	1. Java
	2. Cucumber
	3. Selenium WebDriver
	4. Maven

Required Plug-ins
-----------------
1. Cucumber


To Execute:
----------
To execute the scenarios, if you are using the eclipse IDE and you have all the necessary plug-ins installed
simply goto "Feature" folder, right click on feature file and run as cucumber feature.
